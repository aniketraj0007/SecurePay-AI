API Access & Integration

SecurePay-AI provides a RESTful API for real-time fraud detection. Integrate with your e-commerce or fintech backend in minutes.

Endpoint: POST /check-fraud

POST https://api.securepay-ai.com/v1/check-fraud

Request Body

{
  "amount": 45000,
  "currency": "INR",
  "location": "Mumbai, IN",
  "device_id": "mob_92834",
  "merchant_id": "merch_001"
}

cURL Example

curl -X POST https://api.securepay-ai.com/v1/check-fraud \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "amount": 45000, "location": "Mumbai, IN" }'

Response Schema

{
  "status": "FRAUD",
  "risk_score": 89.4,
  "reasoning": [
    "Unusual velocity detected for merchant_id",
    "High distance from common shipping address"
  ],
  "action": "BLOCK"
}
