### 🌟 The Problem
With the massive surge in digital payments, sophisticated fraud attacks are causing millions in chargeback losses. Startups and SMBs often lack access to enterprise-grade, sub-second latency fraud detection tools. 

### 💡 The Solution
We built **SecurePay-AI**, a B2B SaaS dashboard that uses Machine Learning to calculate transaction risk instantly. 
Our distinct advantage lies in **Explainable AI (XAI)**, robust **Custom Rules Engines**, and a fully accessible **API layer**.

### ✨ Features (The 10/10 Investor Pitch)
- 🧠 **Explainable ML Engine:** Not just a block/allow decision. We tell you *why* (e.g. "Velocity Check Failed", "Geolocation Mismatch").
- 📊 **Pro Analytics:** Live-updating charts powered by `Chart.js` tracking Safe vs Fraud flow in real-time.
- 🚦 **Risk Score Gauge:** Immediate visual understanding of platform threat levels.
- 🚨 **Live Alert Subsystem:** Real-time push of critical flagged activities in your network.
- 🛒 **Pricing & Billing:** Fully fledged SaaS subscription plans directly built-in.
- 🎨 **Extreme Premium UI:** Built with Web3-style Neon Glassmorphism, deep space backgrounds, and CSS3 animations.

### 🛠️ Quick Start (Developer Mode)
Since the MVP is purely a high-octane Vanilla JS Single Page Application (SPA), no build tools are currently required to run the simulation!

1. Clone the repository
2. Open `index.html` in any modern Chromium/WebKit browser.
3. Use the integrated `Auth Simulator` to sign up. 
4. Experience the dashboard!
